Запуска Сервиса первый раз
--------------------------
1. Чтобы 1 раз запустить аппликацию необходимо в файле resources/application.yaml установить значение
   ddl-auto: create-drop
   и запустить сервис.
2. После этого остановить сервис и в файле resources/application.yaml установить значение
   ddl-auto: none
